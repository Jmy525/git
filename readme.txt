分布式(区别集中式)版本控制系统，配置每个机器的家门
git config --global user.name "name"
git config --global user,email "email@example.com"

创建版本库
mkdir git
cd    git
pwd   
ls -ah

添加文件到版本库中
git add readme.txt
git commit  -m "本次提交说明"
